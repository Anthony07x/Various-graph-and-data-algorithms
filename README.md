NodeRank.py - Implements a graph node ranking algorithm (similar to PageRank) to evaluate node importance.
ClosestBlackNode.py - Works in tandem with NodeRank to find the shortest path to the nearest designated "black" (target) node in a graph using multi-source traversal.
GNAlgorithm.py - The Girvan-Newman Algorithm for community detection in networks, which progressively removes edges based on betweenness centrality.
PCY.java - The Park-Chen-Yu (PCY) Algorithm, an optimization of the Apriori algorithm for frequent itemset mining that utilizes a hash table to reduce candidate itemset counts.
SimHash.py - Implements SimHash, a locality-sensitive hashing (LSH) algorithm used to generate fingerprints for determining the similarity of text/documents.
SimHashBuckets.py - Extension of the SimHash implementation, organizing fingerprints into hash buckets to quickly find near-duplicate items.
